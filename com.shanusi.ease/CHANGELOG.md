# Changelog

All notable changes to the Shanusi Ease CEP extension.

## [1.1.0] — 2026-07-01
### Added
- **Live preview** — an animated marker rides the ease curve, plus a horizontal
  "puck" strip below the graph, so you can feel the timing before applying.
- **Bounce / Elastic preview** — each generator now shows its motion curve with
  an animated dot, updating live as you change the parameters.
- **Preset gallery** — a thumbnail grid of every preset (built-in and custom);
  click a mini-curve to load it.
- **Panel icon** — light/dark + rollover icons in the Window > Extensions menu.

### Changed
- In-panel branding header with logo.

## [1.0.0] — 2026-07-01
### Added
- Initial CEP extension: HTML/canvas graph editor with smooth drag and scroll
  zoom (no ScriptUI repaint limits).
- Apply / Get easing, Copy / Paste raw easing, in/out/both mode, randomize.
- Presets (built-in + custom, stored locally) with file export/import.
- Bounce and Elastic generators.
- Make Out — reverse an in-animation into an out-animation at the playhead.
